# openGauss-housekeeper

#### Introduction
GaussDB Group Housekeeper (GGHK) is a project developed by Chinasoft International. GGHK manages openGauss database clusters, automatically handles high availability for primary and standby nodes, and offers a unified floating IP address for openGauss clusters. This lets users access openGauss clusters as if it were a single database.
